Reading in numbers from the user:

When using Read() or Readline(), the values read in are being stored as
chars (Read()) or strings (Readline()).  If we want to use these values 
in computations, we need to convert these values to numbers.  The following 
is an example of reading in a number as a string, but then converting it to an int:

using System;
using System.Collections.Generic;
using System.Text;

namespace InputExample
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.Write("Enter a number: ");
            int number=int.Parse(Console.ReadLine());

            Console.WriteLine("Your number is " + number);

            int square=number*number;

            Console.Write("The square of the number entered is " +square);
            Console.Read();
        }
    }
}
